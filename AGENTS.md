## Code Review Rules

When reviewing pull requests:

- Always respond in Simplified Chinese.
- 所有 Review 评论必须使用简体中文。
- 问题描述使用中文。
- 修复建议使用中文。
- 严重级别使用中文：
  - 阻塞
  - 高风险
  - 中风险
  - 建议优化

Focus only on:
- Logic bugs
- Runtime errors
- Security issues
- Performance issues

Ignore:
- Style-only comments
- Formatting suggestions
